BibTeX Merge Machine

> Merge multiple .bib files based on your LaTeX citations

Suppose you've got multiple .bib files in your LaTeX project, and you're looking to generate a single consolidated bib file with all your references, this tool is for you.

Step 1: Upload .aux file

This tells us which citations you used. In Overleaf, click the document button next to file download, then find "Other logs and files". Download the .aux file from there.
